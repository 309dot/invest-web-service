> **버전**: v2.0  
> **작성일**: 2025년 11월 14일  
> **목적**: 전문가가 초보에게 설명하듯 쉽고 실용적인 포트폴리오 분석 제공

---

## 🎯 핵심 문제 정의

### 현재 문제점
```
❌ "이 숫자가 뭘 의미하는지 모르겠어요"
❌ "데이터는 많은데 나한테 무슨 도움이 되는지 모르겠어요"
❌ "그래서 내가 뭘 해야 하는지 모르겠어요"
❌ "너무 많은 컬러와 차트에 눈이 피로해요"
```

### 개선 방향
```
✅ "아, 이게 이런 의미구나!"
✅ "이 지표가 나한테 왜 중요한지 알겠어요"
✅ "다음에 이렇게 해보면 되겠네요"
✅ "한눈에 들어오고 집중이 잘 돼요"
```

---

## 🎓 설계 철학: "전문가의 1:1 코칭"

### 핵심 원칙
1. **스토리텔링**: 숫자 나열이 아닌 이야기로 설명
2. **맥락 제공**: 데이터의 의미를 실생활 예시로 연결
3. **행동 유도**: 데이터를 보고 무엇을 해야 하는지 명확히
4. **점진적 학습**: 기본 → 중급 → 고급으로 단계적 정보 제공
5. **시각적 단순화**: 필요한 정보만 강조, 불필요한 요소 제거
