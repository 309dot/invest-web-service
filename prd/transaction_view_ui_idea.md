# 💡 거래 이력 화면 간소화 아이디어 제안

## 1️⃣ 요약 카드로 데이터 덩어리 줄이기
현재 페이지 상단은 “총 매수 금액 / 매도 금액 / 순매수”로 단순 텍스트만 표기되어 있음.  
→ 시각적 카드 요약 구조로 변경 추천.

**예시**
```tsx
<SummaryCards>
  <Card title="총 매수 금액" value="$944.57" sub="4.71주 / 평균 $200.70" />
  <Card title="총 매도 금액" value="$195.04" sub="0.47주 / 평균 $412.69" />
  <Card title="순매수" value="- $749.52" tone="negative" />
</SummaryCards>
```
✅ 한눈에 자산 흐름 인식 가능  
✅ 음수/양수 색상 강조 (손실은 붉은톤)  
✅ 모바일에서도 스와이프로 확인 가능  

---

## 2️⃣ 종목별 그룹 뷰 제공
현재는 **61개 거래가 모두 타임라인 형태로 나열**되어 있음 → 가독성이 매우 낮음.  
→ **종목 기준 그룹화 + 접기/펼치기 구조**로 변경.

```tsx
<Accordion title="MGK (최근 30회)">
  <TransactionItem date="2025-10-30" type="매수" amount="$5.00" />
  ...
</Accordion>
```
✅ 같은 종목은 묶어 보기  
✅ 종목 옆에 “+7.2% 수익중” 같은 요약 표시  
✅ 접기/펼치기로 불필요한 거래 숨김  

---

## 3️⃣ 자동투자 / 수동거래 구분 탭
현재 “자동 투자 (daily)”와 “추가 매수(병합)”이 혼재되어 있음.  
→ 필터 탭 분리로 개선.

```
[전체] [자동 투자] [수동 매수/매도]
```
✅ 자동 투자 내역만 빠르게 점검 가능  
✅ 수동 거래만 보면 내가 직접 한 매매 흐름 파악 용이  

---

## 4️⃣ 거래 요약 타임라인 뷰 추가
스크롤형 표보다, 일 단위/주 단위 요약 그래프를 추가.

**예시**
- 📆 10월 마지막 주: 5회 매수, 1회 매도 (순매수 $-120)
- 📆 이번 주: 2회 매수, 0회 매도 (+$35)

→ 주간 리포트와 연동 시 “투자 패턴 인사이트”로 확장 가능.

---

## 5️⃣ 금액 / 단가 표시 단순화
현재 “$5.00 / $5.00 / 총 금액 $5.00”처럼 중복 표현이 많음.  
→ **표시 규칙 간소화**:

```
매수 0.0118주 @ $422.94 → $5.00
```
✅ 직관적이고 표 크기 절반으로 감소  
✅ 거래 로그 UI가 훨씬 깔끔해짐  

---

## 6️⃣ “요약 → 상세” 구조로 정보 계층 재편
현재는 모든 거래가 한 수준에 존재.  
→ ‘빠르게 훑고, 필요 시 상세 확인’ 구조로 변경.

**2단 구성 제안**
- 리스트: 날짜 / 종목 / 거래유형 / 금액  
- 클릭 시 상세: 가격, 수수료, 메모 표시

```tsx
<TransactionListItem>
  <span>2025-10-30</span>
  <span>MGK</span>
  <span>매도</span>
  <span>$55.83</span>
</TransactionListItem>
```
✅ 클릭 시 팝오버로 상세 내역 표시
