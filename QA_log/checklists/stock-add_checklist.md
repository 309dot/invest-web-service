## 🧪 기능 테스트

### 정상 플로우
- [ ] **한국 주식 추가**: 삼성전자 10주, 70,000원에 구매
  - 검색 > 선택 > 정보 입력 > 저장 > 포트폴리오에 표시 확인
- [ ] **미국 주식 추가**: Apple 5주, $180에 구매
  - 검색 > 선택 > 정보 입력 > 환율 적용 확인 > 저장
- [ ] **ETF 추가**: SCHD 3주, $80에 구매
  - ETF로 정상 인식되는지 확인
- [ ] **자동 구매 설정**: QQQ 매월 $500씩 자동 구매
  - 반복 구매 일정이 생성되는지 확인

### 에러 핸들링
- [ ] 종목 미선택 시 에러 메시지
- [ ] 수량 입력 안 함 (또는 0) 시 에러 메시지
- [ ] 수량에 음수 입력 시 에러 메시지
- [ ] 가격 입력 안 함 (또는 0) 시 에러 메시지
- [ ] 가격에 음수 입력 시 에러 메시지
- [ ] 미래 날짜 선택 시 에러 메시지
- [ ] 중복 종목 추가 시 확인 메시지 (또는 수량 추가 제안)

### 경계값 테스트
- [ ] 수량 최솟값: 0.0001주 (소수점 지원)
- [ ] 수량 최댓값: 1,000,000주
- [ ] 가격 최솟값: $0.01 또는 1원
- [ ] 가격 최댓값: $1,000,000 또는 10억원
- [ ] 매우 긴 종목명 (50자 이상)
- [ ] 특수문자 포함 종목명

### 데이터 검증
- [ ] 저장 후 데이터베이스에 올바르게 저장되었는지 확인

- [ ] 포트폴리오 총액에 정확히 반영되는지 확인
